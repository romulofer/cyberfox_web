# cyberfox_web

## Visão Geral do Projeto

gere a descrição desse projeto com base no repositório presente em ~/development/x_platform_apps/cyberfox. este projeto deve ser uma versão web idêntica ao cyberfox. deve ser usado svelte e sveltekit, com bun e typescript. 

## Tech Stack

| Categoria | Tecnologia | Versão / Notas |
|----------|------------|-----------------|
| frontend | svelte, sveltekit, bun | latest |
| deployment | docker, docker compose | irei fazer o deploy desse projeto no github pages, mas o usuário pode rodar uma versão self hosted |

- pode ser self hosted
